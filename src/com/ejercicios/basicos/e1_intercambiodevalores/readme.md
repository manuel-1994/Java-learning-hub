# Ejercicio nº 1 - Intercambio de valores

Realizar un programa que permita el intercambio de valores entre dos variables.

Por ejemplo: Si una variable __numero__ vale 35 y una varibale __numero2__ vale 20, realizar las acciones necesarias para que **numero** pase a valer 20 y **numero2** pase a valer 35. Una vez realizado el cambio mostrar el resultado por pantalla.
