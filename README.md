# 🚀 Mi Hub de Aprendizaje de Java

¡Bienvenido a mi portafolio y bitácora de desarrollo en Java! Este espacio está diseñado como un ecosistema completo donde documento la base teórica del lenguaje y la pongo en práctica inmediatamente a través de ejercicios y proyectos estructurados.

---

## 📂 Arquitectura del Proyecto

Para mantener un entorno limpio y profesional, el repositorio está dividido rigurosamente en dos secciones independientes:

* **[📚 Sección Teórica](./teoria/index.md):** Apuntes, conceptos clave, diagramas de arquitectura y resúmenes técnicos organizados cronológicamente por módulos.
* **[💻 Código Fuente (`src`)](./src/com/ejercicios/):** Soluciones reales programadas en Java.

---

## ⚙️ Tecnologías y Herramientas Utilizadas
* **Lenguaje:** Java (JDK 17 o superior)
* **IDE Recomendado:** IntelliJ IDEA / NetBeans / VS Code
* **Documentación:** Markdown renderizado con GitHub Pages
* **Control de Versiones:** Git y GitHub
